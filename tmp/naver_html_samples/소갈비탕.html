        갈비는 넉넉한 크기의 그릇에 담아 물을 붓고 핏물을 빼주는데요. 삼투압을 이용해 효과적으로 빼주기 위해 설탕 등을 이용하기도 하요. 오랜 시간 담가주면 맛 성분까지 빠질 수 있으니까요. 오늘은 설탕 대신 단맛이 있는 '갈아 만든 배'를 사용했어요. 사진은 4kg이고 그중 3kg 사용했어요. 갈아만든배 1리터를 붓고 나
        2. 갈비 손질하기
        두툼한 그대로 사용해도 되지만 도톰한 살을 포 떠서 칼집을 넣어 손질해 줍니다. 불필요한 지방은 제거해 주고요.
        3. 데치기
        고기가 잠길 정도의 물을 붓고 양파, 생강, 월계수잎을 넣어 끓여주어요. 물이 팔팔 끓으면 손질한 갈비를 넣고 10분 정도 데쳐줍니다. 떠오르는 거품은 걷어내며 데쳐주고요.
        4. 세척하기
        데친 후 흐르는 물에 하나하나 세척해 주어요. 특히 뼈 쪽에 피떡이 굳으며 뭉친 게 있으니 잘 닦아내어주고요. 살 사이사이 찌꺼기 굳은 게 있다면 제거하며 세척해 줍니다. 세척하여 체에 담아 물기를 제거해 놓아요.

        갈비탕 끓이는법 소갈비탕 만드는법
        1.갈비탕 재료
        갈비탕 계량 1T = 15ml = 15cc 1t = 5ml = 5cc 1C(1컵) = 200ml = 200cc 갈비찜용 1kg / 데칠때 월계수잎 2장, 통후추 조금, 청주 2T 대파 1대, 무 1토막, 대추 한줌, 다시마 손바닥크기 3장 무 250g, 대파 1/2대, 국간장 3T, 다진마늘 1T, 후추 조금
        2.갈비 핏물 제거하기
        갈비는 찜갈비용을 사용했어요. 찬물에 갈비를 넣어 주고 설탕을 조금 넣어 여러번 번갈아 가며 핏물을 빼 주었어요.
        3.갈비 데치기
        핏물을 뺀 갈비는 냄비에 넣고 물을 부어 주고 월계수잎과 통후추를 넣어 주고 청주를 조금 부어 줍니다. 갈비에서 핏물 거품과 뼈가루들이 떠 오르고 살짝 데쳐지면 찬물에 여러번 깨끗하게 씻어 줍니다.
        4. 갈비 칼집내어 주기
        깨끗하게 씻은 갈비는 칼집을 내어 줍니다. 이렇게 하면 갈비를 부드럽게 먹을 수 있고 양념이 스며들어 맛있어요. 그리고 옆에 지방이 있다면 일부는 잘라 주었답니다.
